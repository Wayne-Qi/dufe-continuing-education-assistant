# 🎓 东北财经大学继续教育在线学习助手

> 课程自动播放助手 - 全自动循环学习

## ✨ 功能特点

- 🎬 **自动播放**：进入课程后自动播放视频
- 🔁 **自动循环**：章节完成后自动进入下一章
- 📊 **进度监控**：实时显示播放进度和剩余时间
- 🔇 **静音播放**：自动静音，不影响其他操作
- 💬 **弹窗处理**：自动关闭课程评价弹窗
- 📋 **仪表盘面板**：可视化显示当前状态

1. 脚本猫 https://scriptcat.org/zh-CN/script-show-page/6844

## 🚀 使用说明

1. https://kjw.hlj.gov.cn/jxjy/jxjycx
2. 选择东北财经大学继续教育平台
3. 进入「课程学习」页面
4. 脚本会自动识别并进入未完成的课程
5. 视频自动播放，完成后自动返回列表
